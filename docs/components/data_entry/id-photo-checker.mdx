# IdPhotoChecker 证件照检测

基于 [@vladmandic/human](https://github.com/vladmandic/human) 的实时证件照拍摄检测组件，通过摄像头实时检测人脸，支持边框偏离检测、戴帽检测、戴眼镜检测、面部遮挡检测等，适用于证件照拍摄场景。

## 基本用法

```tsx
import { IdPhotoChecker } from "zs_library";

export default () => {
  return (
    <IdPhotoChecker
      onCapture={(imageData, result) => {
        console.log("拍照结果", imageData, result);
      }}
    />
  );
};
```

## 自定义验证规则

可通过 `rules` 选择性开启或关闭某些检测项：

```tsx
import { IdPhotoChecker } from "zs_library";

export default () => {
  return (
    <IdPhotoChecker
      rules={{
        checkBounds: true,
        checkHat: true,
        checkGlasses: false,
        checkOcclusion: true,
        checkOrientation: true,
      }}
      onCapture={(imageData) => {
        console.log("拍照完成", imageData);
      }}
    />
  );
};
```

## 自定义提示文案

通过 `messages` 可以覆盖默认的中文提示：

```tsx
import { IdPhotoChecker } from "zs_library";

export default () => {
  return (
```

## 使用 Hook

如果需要完全自定义 UI，可以直接使用 `useIdPhotoChecker` Hook：

```tsx
import { useIdPhotoChecker } from "zs_library";

export default () => {
  const {
    videoRef,
    validation,
    isLoading,
    isCameraReady,
    error,
    capture,
    start,
    stop,
  } = useIdPhotoChecker({
    autoStart: true,
    onValidationChange: (result) => {
      console.log("验证结果变化", result);
    },
  });

  return (
    <div>
      <video ref={videoRef} autoPlay playsInline muted />
      {isLoading && <p>加载中...</p>}
      {error && <p>错误：{error.message}</p>}
      {isCameraReady && (
        <div>
          <p>检测到 {validation.faceCount} 张人脸</p>
          <p>是否通过：{validation.isValid ? "是" : "否"}</p>
          <ul>
            {validation.items.map((item) => (
              <li key={item.key}>
                {item.key}: {item.status} - {item.message}
              </li>
            ))}
          </ul>
          <button onClick={capture} disabled={!validation.isValid}>
            拍照
          </button>
        </div>
      )}
    </div>
  );
};
```

## 调整检测阈值

通过 `thresholds` 微调各检测项的灵敏度：

```tsx
import { IdPhotoChecker } from "zs_library";

export default () => {
  return (
    <IdPhotoChecker
      thresholds={{
        minFaceScore: 0.6,
        minBoundsOverlap: 0.75,
        hatHeadRatio: 0.18,
        glassesZThreshold: 0.025,
        minMeshScore: 0.85,
        maxYawAngle: 12,
        maxPitchAngle: 12,
      }}
      onCapture={(imageData) => {
        console.log(imageData);
      }}
    />
  );
};
```

## 检测能力

| 检测项 | 说明 | 原理 |
| --- | --- | --- |
| 边框检测 | 检测人脸是否在椭圆引导框内 | 计算人脸边界框与引导框的重叠比例 |
| 戴帽检测 | 检测是否佩戴帽子 | 分析额头上方区域占比（眉间到 face box 顶部的比例） |
| 眼镜检测 | 检测是否佩戴眼镜 | 眼部关键点 z 轴偏移与面颊/鼻尖参考点对比 |
| 遮挡检测 | 检测面部是否有遮挡 | 面部 mesh 置信度分数低于阈值 |
| 朝向检测 | 检测是否正面朝向镜头 | yaw/pitch 角度 + gesture 手势识别辅助 |

## 特性

- 📷 **实时检测**: 基于 `@vladmandic/human` 实现毫秒级人脸检测
- 🎯 **多维度校验**: 边框、帽子、眼镜、遮挡、朝向五大维度
- 💬 **中文提示**: 默认提供中文检测提示，支持自定义文案
- 🎛️ **高度可配**: 检测规则、阈值、边框、摄像头参数均可自定义
- 🪝 **Hook 支持**: 提供 `useIdPhotoChecker` 用于完全自定义 UI
- 🔧 **调试模式**: 开启 `debug` 可查看实时检测数据
- 📐 **TypeScript**: 完整类型定义

## API

### IdPhotoChecker

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 自定义类名 | `string` | - |
| width | 容器宽度 | `number` | `400` |
| height | 容器高度 | `number` | `500` |
| rules | 验证规则配置 | `ValidationRuleConfig` | 全部开启 |
| bounds | 椭圆引导框位置配置 | `BoundsConfig` | `{ left: 0.2, top: 0.1, width: 0.6, height: 0.75 }` |
| thresholds | 检测阈值配置 | `ThresholdConfig` | 见下方 |
| camera | 摄像头配置 | `CameraConfig` | `{ facingMode: 'user', width: 640, height: 480 }` |
| messages | 提示文案配置 | `MessageConfig` | 默认中文提示 |
| modelBasePath | Human 模型文件路径 | `string` | CDN 路径 |
| detectInterval | 检测间隔（ms） | `number` | `200` |
| debug | 是否显示调试面板 | `boolean` | `false` |
| autoStart | 是否自动开启摄像头 | `boolean` | `true` |
| guideBorderPassColor | 引导框通过时颜色 | `string` | `'#4ade80'` |
| guideBorderFailColor | 引导框未通过时颜色 | `string` | `'#f87171'` |
| loadingComponent | 自定义加载态组件 | `ReactNode` | 默认 Loading |
| onValidationChange | 验证结果变化回调 | `(result: ValidationResult) => void` | - |
| onCapture | 拍照回调 | `(imageData: string, result: ValidationResult) => void` | - |
| onReady | 模型加载就绪回调 | `() => void` | - |
| onError | 错误回调 | `(error: Error) => void` | - |

### ValidationRuleConfig

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| checkBounds | 是否启用边框检测 | `boolean` | `true` |
| checkHat | 是否启用帽子检测 | `boolean` | `true` |
| checkGlasses | 是否启用眼镜检测 | `boolean` | `true` |
| checkOcclusion | 是否启用遮挡检测 | `boolean` | `true` |

### ThresholdConfig

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| minFaceScore | 最小人脸置信度 | `number` | `0.5` |
| minBoundsOverlap | 最小边框重叠度 | `number` | `0.7` |
| hatHeadRatio | 帽子检测阈值（额头占比） | `number` | `0.15` |
| glassesZThreshold | 眼镜检测 z 轴偏移阈值 | `number` | `0.02` |
| minMeshScore | 遮挡检测最低 mesh 分数 | `number` | `0.8` |
| maxYawAngle | 最大左右偏转角度（度） | `number` | `15` |
| maxPitchAngle | 最大上下俯仰角度（度） | `number` | `15` |

### CameraConfig

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| facingMode | 摄像头方向 | `'user' \| 'environment'` | `'user'` |
| width | 视频宽度 | `number` | `640` |
| height | 视频高度 | `number` | `480` |

### BoundsConfig

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| left | 左边距比例（0-1） | `number` | `0.2` |
| top | 上边距比例（0-1） | `number` | `0.1` |
| width | 宽度比例（0-1） | `number` | `0.6` |

### useIdPhotoChecker

Hook 接受与组件相同的配置参数（除 UI 相关属性），返回：

| 返回值 | 说明 | 类型 |
| --- | --- | --- |
| videoRef | 视频元素引用 | `RefObject<HTMLVideoElement>` |
| canvasRef | Canvas 元素引用 | `RefObject<HTMLCanvasElement>` |
| validation | 当前验证结果 | `ValidationResult` |
| isLoading | 模型是否加载中 | `boolean` |
| isCameraReady | 摄像头是否就绪 | `boolean` |
| error | 错误信息 | `Error \| null` |
| start | 手动开始检测 | `() => Promise<void>` |
| stop | 手动停止检测 | `() => void` |
| capture | 拍照（返回 base64） | `() => string \| null` |

### ValidationResult

```typescript
interface ValidationResult {
  isValid: boolean;
  items: ValidationItem[];
  faceCount: number;
  faceScore: number;
}

interface ValidationItem {
  key: string;
  status: "pass" | "fail" | "warning" | "pending";
  message: string;
}
```

## 注意事项

1. **浏览器兼容性**: 需要支持 `getUserMedia` API 的现代浏览器（Chrome 53+、Firefox 36+、Safari 11+、Edge 12+）
2. **HTTPS 要求**: 大多数浏览器要求在 HTTPS 环境下才能访问摄像头
3. **模型加载**: 首次使用会从 CDN 加载 `@vladmandic/human` 模型文件（约 10MB），后续会被浏览器缓存
4. **性能**: 建议 `detectInterval` 不低于 150ms，过低可能导致低端设备卡顿
5. **帽子/眼镜检测**: 基于关键点几何推断，非专用分类模型，存在一定误差率，可通过调整 `thresholds` 优化
